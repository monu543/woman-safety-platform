# 👩‍💻 Women Safety Platform

## 📌 Overview
The **Women Safety Platform** is a web-based application designed to enhance women's safety by providing emergency assistance features such as live location tracking, SOS alerts, and emergency contact management. This platform enables users to quickly seek help during emergencies and ensures better safety through technology.

---

## 🚀 Features

- 🔐 **User Authentication** – Secure Login & Registration using JWT.
- 📍 **Live Location Tracking** – Share real-time location using interactive maps.
- 🚨 **SOS Emergency Alert** – Send instant alerts to emergency contacts.
- 📞 **Emergency Contacts Management** – Add and manage trusted contacts.
- 🗺️ **Interactive Map Integration** – Implemented using Leaflet/Google Maps.
- 📊 **User Dashboard** – Centralized interface to manage safety features.
- ☁️ **Cloud Database** – Secure data storage with MongoDB Atlas.

---

## 🛠️ Tech Stack

### 🎨 Frontend
- React.js
- HTML5, CSS3, JavaScript (ES6+)
- React Router DOM
- Axios
- Leaflet / Google Maps

### 🖥️ Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- JSON Web Token (JWT)
- bcrypt.js for password hashing
- CORS & dotenv
